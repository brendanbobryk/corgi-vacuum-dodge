A small Frogger-inspired React game where you guide a corgi safely across the floor while avoiding roaming vacuums to reach toys and treats at the top!  
This project was built as part of a daily game challenge to practice React fundamentals, game loops, collision detection, and grid-based movement.

## Features

- Grid-based movement using arrow keys
- Moving vacuum obstacles with lane-based speeds
- Collision detection that resets the corgi on contact
- Goal row filled with toys, plushies, and treats
- Score tracking for successful crossings
- Dark-themed, easy-on-the-eyes layout
- Simple, responsive game board

## Technologies Used

- React (with hooks)
- JavaScript
- CSS

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/brendanbobryk/corgi-vacuum-dodge.git

2. Navigate to the project folder:
    ```bash
    cd corgi-vacuum-dodge

3. Install dependencies:
    ```bash
    npm install

4. Start the development server:
    ```bash
    npm run dev

5. Open your browser at the local address shown in the terminal (usually http://localhost:5173) to play!
